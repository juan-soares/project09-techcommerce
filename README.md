## Description

Projeto para prática de desenvolvimento web Front-end, utilizando HTML, CSS e JavaScript. Consiste em um E-Commerce ficiticio que comercializa produtos representando as áreas do desenvolvimento web. Contém uma lista de produtos que são renderizados dinamicamente na página.

## Features

- Favicon,
- Modularização de arquivos JavaScript;
- Ícones de biblioteca externa via CDN (Font Awesome);
- Renderização de conteúdo na página através de JavaScript;
- Lista lateral de itens no Carrinho, com visibilidade é controlada por botões;
- Design Responsivo.

## How to use

- `Step 1`: Baixar os arquivos do projeto.
- `Step 2`: Executar o arquivo "index.html" em qualquer navegador.
